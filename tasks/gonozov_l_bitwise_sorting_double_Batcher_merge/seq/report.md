Цель — сортировка массива чисел типа `double` с использованием комбинации поразрядной сортировки

## 2. Постановка задачи

- **Вход**: `std::vector<double>` произвольной длины
- **Выход**: отсортированный по возрастанию `std::vector<double>`
- **Ограничения**: массив не должен быть пустым (проверка в `ValidationImpl`)
- **Крайние случаи**: пустой массив, массив из одного элемента, массив с отрицательными числами

## 3. Базовый алгоритм

### Поразрядная сортировка для double

Проблема: битовое представление `double` не позволяет напрямую использовать целочисленную 
radix sort (отрицательные числа идут после положительных).

**Решение** — битовое преобразование:

1. Для положительных чисел: установить старший бит в 1
2. Для отрицательных чисел: инвертировать все биты
3. После сортировки — обратное преобразование

### Слияние Бэтчера

Рекурсивная сеть слияния двух отсортированных массивов. Для итеративной реализации:

- Сначала сливаются подмассивы длины 1 → 2 → 4 → ... → n
- В каждом блоке выполняется `log2(len)` проходов с уменьшающимся шагом

### Общая схема HybridSortDouble

1. Дополнение массива до ближайшей степени двойки
2. Разделение на две половины (левая и правая)
3. Поразрядная сортировка каждой половины
4. Слияние половин сетью Бэтчера
5. Обрезание до исходного размера

**Асимптотика**:

- Время: O(n·log n) — radix sort O(8n) + Batcher merge O(n·log²n)
- Память: O(n) — дополнительные массивы для radix sort

// Преобразование double -> uint64_t для сортировки
uint64_t DoubleToSortableInt(double d) {
  uint64_t bits;
  std::memcpy(&bits, &d, sizeof(double));
  if ((bits >> 63) != 0) return ~bits;      // отрицательное
  return bits | 0x8000000000000000ULL;      // положительное
}

// Поразрядная сортировка (8 проходов, radix = 256)
void RadixSortDouble(std::vector<double>& data);

// Слияние половинок для сети Бэтчера
