## 1. Контекст

Данный отчёт описывает последовательную (базовую) версию умножения разреженных
матриц в формате CRS. Эта реализация служит эталоном корректности и
производительности (baseline) для всех параллельных версий (OMP, TBB, STL, ALL).
Алгоритм включает транспонирование матрицы B для оптимизации доступа по столбцам
и последующее построчное умножение.

## 2. Постановка задачи

- **Вход:** Матрицы A и B в CRS формате.
- **Выход:** C = A × B в CRS формате.
- **Ограничения:** A.ncols == B.nrows. |value| < 10⁻⁵ → элемент считается
  нулевым.

## 3. Базовый алгоритм

1. **Транспонирование B → Bᵀ**:
   - Подсчёт числа элементов в каждой строке `Bᵀ` (проход по B.cols)
   - Построение row_ptrs для Bᵀ через префиксные суммы
   - Заполнение values и cols для Bᵀ

2. **Умножение** (для каждой строки i матрицы A):
   - Инициализация маркерного массива `temp[ncols(A)]` = -1
   - Для каждого ненулевого элемента `A[i][k]`: `temp[k]` = позиция
   - Для каждого столбца j (строка j в Bᵀ):
     - sum = 0
     - Для каждого ненулевого элемента `Bᵀ[j][k]`:
       - Если `temp[k]` != -1: sum += A_val \* B_val
     - Если |sum| ≥ 10⁻⁵: добавить в результат

- **Сложность по времени**: O(nz(A) + nz(B) + nz(C))
- **Сложность по памяти**: O(nz(A) + nz(B) + nz(C) + P × ncols(A))
- **Инварианты**: `row_ptrs[0]=0`, `row_ptrs[nrows]=nz`, индексы столбцов в
  `[0,  ncols)`

## 4. Детали реализации

- **Файлы**: `seq/include/ops_seq.hpp`, `seq/src/ops_seq.cpp`
- **Валидация (`ValidationImpl`)**: A.ncols == B.nrows, nz ≥ 0
- **Предобработка (`PreProcessingImpl`)**: Копирование входных данных
- **Запуск (`RunImpl`)**: Вызов `MultiplierSeq::Multiply`
- **Постобработка (`PostProcessingImpl`)**: Копирование результата

**Ключевой фрагмент (маркерный массив и умножение):**

```cpp
// Файл: seq/src/multiplier_seq.cpp
std::vector<int> temp(a.ncols, -1);
for (size_t idx = row_start; idx < row_end; ++idx) {
    temp[a.cols[idx]] = idx;
}
for (size_t j = 0; j < bt.nrows; ++j) {
    double sum = 0.0;
    size_t bt_start = bt.row_ptrs[j];
    size_t bt_end = bt.row_ptrs[j + 1];
    for (size_t idx = bt_start; idx < bt_end; ++idx) {
        int k = bt.cols[idx];
        if (temp[k] != -1) {
            sum += a.values[temp[k]] * bt.values[idx];
        }
    }
    if (std::abs(sum) >= kZERO) {
        // добавление в результат
    }
}
```

## 5. Проверка корректности

- **Сравнение с эталоном**: функция `Equal` (порог 10⁻⁵) для каждого тестового
  файла
- **Крайние случаи**: пустые матрицы (nz=0), 1×1 матрицы, прямоугольные матрицы
- **Инварианты**: проверка row_ptrs, диапазонов индексов
- **Все тесты пройдены**: 1×sparse×dense, 1×dense×sparse, 4×sparse×sparse

Последовательная версия является корректным и надёжным эталоном для оценки
