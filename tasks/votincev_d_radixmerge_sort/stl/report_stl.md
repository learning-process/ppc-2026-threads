## 1. Введение

Сортировка данных — фундаментальная задача программирования. Поразрядная сортировка (Radix Sort) работает
за линейное время относительно размера входных данных при фиксированном диапазоне чисел. Для её ускорения
можно использовать встроенные средства многопоточности стандарта C++, такие как `std::async` и `std::future`,
которые позволяют абстрагироваться от явного создания системных потоков и автоматически распределять задачи.

Данная работа посвящена реализации параллельной версии алгоритма поразрядной сортировки с использованием
средств многопоточности STL и анализу её эффективности.

## 2. Постановка задачи

Необходимо реализовать параллельный алгоритм поразрядной сортировки (LSD) для вектора 32-битных знаковых
целых чисел с использованием возможностей стандартной библиотеки C++.

- **Входные данные**: Вектор `std::vector<int32_t>`.
- **Выходные данные**: Отсортированный вектор.
- **Ограничения**: Поддержка отрицательных чисел, использование `std::async` для параллельного выполнения
  задач слияния и сортировки.

## 3. Описание алгоритма (последовательный)

Последовательный алгоритм служит базой для локальной сортировки. Его шаги:

1. Определение минимального и максимального элементов в векторе для вычисления диапазона.
2. Приведение чисел к положительному диапазону (сдвиг на величину $|min\_val|$).
3. Применение десятичной поразрядной сортировки (Counting Sort для каждого разряда).
4. Обратный сдвиг значений для восстановления исходных чисел.

## 4. Схема распараллеливания

В STL-версии распараллеливание реализовано на основе асинхронных задач:

1. **Разделение**: Входной массив разделяется на $N$ блоков.
2. **Асинхронная локальная сортировка**: Для каждого блока через `std::async(std::launch::async, ...)`
   запускается отдельная задача, выполняющая локальный Radix Sort.
3. **Параллельное слияние (Tree Merge)**: Отсортированные блоки объединяются попарно. На каждом шаге
   независимые пары объединяются в `std::async`. После этого результат выполнения дожидается через `f.get()`,
   и процесс продолжается на следующем уровне дерева слияния.

## 5. Детали реализации

- **Класс**: `VotincevDRadixMergeSortSTL`.
- **Локальная сортировка**: Метод `LocalRadixSort` сортирует один из блоков данных.
- **Слияние**: Функция `Merge` производит объединение соседних локальных участков с использованием
  временного буфера.
- **Синхронизация**: Синхронизация потоков на каждом этапе слияния обеспечивается вызовом метода `.get()`
  у объектов `std::future`.
- **std::launch::async**: гарантирует создание потока, а не отложенное выполнение.

## 8. Заключение

Использование `std::async` из библиотеки STL позволило эффективно распределить вычисления. Время выполнения
сортировки сократилось с 2.414с до 1.280с на 4 потоках. Полученное ускорение (~1.89x) сопоставимо с результатами
OpenMP и TBB. Снижение эффективности при масштабировании связано с издержками на запуск асинхронных задач
и операциями копирования при слиянии подмассивов.

## Приложение

```cpp
// Если массив пустой - сортировать нечего
bool VotincevDRadixMergeSortSTL::ValidationImpl() {
  return !GetInput().empty();
}

// Препроцессинга нет для сортировки
// (для изображений могло бы быть что-то вроде перевода в ч/б)
bool VotincevDRadixMergeSortSTL::PreProcessingImpl() {
  return true;
}

// Постобработка не проводится
bool VotincevDRadixMergeSortSTL::PostProcessingImpl() {
  return true;
}

bool VotincevDRadixMergeSortSTL::RunImpl() {
  const auto &input = GetInput();
  auto n = static_cast<int32_t>(input.size());
  if (n == 0) {
    return true;
  }

  int32_t min_val = *std::ranges::min_element(input.begin(), input.end());

  std::vector<uint32_t> working_array(static_cast<size_t>(n));
  for (int32_t i = 0; i < n; ++i) {
    working_array[static_cast<size_t>(i)] =
        static_cast<uint32_t>(input[static_cast<size_t>(i)]) - static_cast<uint32_t>(min_val);
  }

  std::vector<uint32_t> temp_buffer(static_cast<size_t>(n));
  ParallelRadixMergeSort(working_array.data(), n, temp_buffer.data());

  std::vector<int32_t> result(static_cast<size_t>(n));
  for (int32_t i = 0; i < n; ++i) {
    result[static_cast<size_t>(i)] =
        static_cast<int32_t>(working_array[static_cast<size_t>(i)] + static_cast<uint32_t>(min_val));
  }

  GetOutput() = std::move(result);
  return true;
}
