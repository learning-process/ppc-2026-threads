## 1. Контекст

Данная работа реализует умножение разреженных матриц с использованием ручного управления
потоками через `std::thread`. Это позволяет:

- Изучить низкоуровневые аспекты многопоточного программирования
- Сравнить ручное управление потоками с высокоуровневыми подходами (OpenMP, TBB)
- Понять преимущества и недостатки автоматических систем балансировки
Последовательная версия (`seq/report.md`) служит эталоном корректности, а OpenMP/TBB — для
сравнения с автоматизированными подходами.

## 2. Постановка задачи

### Входные данные

Две разреженные матрицы A и B в формате CCS.
Структура SparseMatrixCCS:

- `values` — массив ненулевых значений (тип double);
- `row_indices` — массив индексов строк для каждого значения;
- `col_ptrs` — массив указателей на начало каждого столбца (размер `cols + 1`);
- `rows` — количество строк;
- `cols` — количество столбцов.

### Выходные данные

Результирующая матрица C = A × B в формате CCS.

### Ограничения

- Количество столбцов матрицы A должно равняться количеству строк матрицы B (причем количество
строк и столбцов матрицы должно быть неотрицательно);
- Все индексы строк находятся в диапазоне `[0, rows-1]`;
- Указатели `col_ptrs` являются неубывающей последовательностью
- Первый указатель `col_ptrs[0] = 0`;
- Последний указатель `col_ptrs[cols] = общему количеству ненулевых элементов`.

## 3. Базовый алгоритм

Последовательная версия вычисляет произведение матриц по следующей схеме. Для каждого столбца
j матрицы B:

- Обход ненулевых элементов столбца j, получение пар (k, `B[k][j]`);
- Для каждого k обход ненулевых элементов столбца k матрицы A;
- Накопление в `temp[i]: A[i][k] × B[k][j]`;
- Сохранение ненулевых значений из temp в результирующую матрицу.
Детальное описание последовательного алгоритма приведено в `seq/report.md`.

## 4. Схема распараллеливания

В отличие от OpenMP и TBB, где распределение работы автоматическое, в STL-версии используется
статическое разбиение:

```cpp
const unsigned int hardware_threads = std::thread::hardware_concurrency();
const int num_threads = (hardware_threads > 0) ? static_cast<int>(hardware_threads) : 1;
const int cols_per_thread = (b.cols + num_threads - 1) / num_threads;
```

### Схема разбиения

- Определяется аппаратное количество потоков через `hardware_concurrency()`;
- Столбцы матрицы B делятся на примерно равные блоки;
- Каждый поток обрабатывает непрерывный диапазон столбцов `[start_col, end_col)`;
- Последний поток может получить меньше столбцов (при неделимости).

### Память и синхронизация

- Нет гонок данных — каждый поток записывает в разные позиции массива col_start (разные j);
- Нет атомарных операций — запись в непересекающиеся области памяти безопасна;
- Явные барьеры — join() в конце каждого прохода обеспечивает синхронизацию.

### Двухпроходный алгоритм

Как и в OpenMP/TBB версиях, алгоритм разделён на два прохода:

1. Первый проход (подсчёт) — вычисление количества ненулевых элементов в каждом столбце результата.
2. Второй проход (заполнение) — запись значений в заранее выделенные массивы.

## 5. Детали реализации

Файлы: `stl/include/ops_stl.hpp`, `stl/src/ops_stl.cpp`

### Основные методы

`ValidationImpl()` — проверка корректности входных данных:

- Валидность структуры CCS обеих матриц
- Совместимость размерностей (A.cols == B.rows)

`PreProcessingImpl()` — подготовка к вычислениям:

- Создание результирующей матрицы C с размерами A.rows × B.cols

`RunImpl()` — выполнение умножения:

- Вызов `MultiplyMatrices(A, B)`

`PostProcessingImpl()` — завершающие действия:

- В данной реализации не требуется (возвращает true)

### Вспомогательные методы

`IsMatrixValid()` — проверка валидности CCS формата (реализация ограничений из 2):

- Размеры матрицы неотрицательны;
- Размер `col_ptrs` равен `cols + 1`;
- Количество значений равно количеству индексов строк;
- Первый указатель `col_ptrs[0] = 0`;
- Последний указатель `col_ptrs[cols] = nnz` (где `nnz(X)` — количество ненулевых элементов в
матрице X);
- Указатели неубывают и неотрицательны;
- Индексы строк в допустимом диапазоне.

`ComputeColumn()` — вычисляет значения одного столбца результирующей матрицы (листинг представлен
в файле `omp/report.md`).

`CountNonZero()` — подсчитывает количество ненулевых элементов в столбце с заданной погрешностью
(листинг представлен в файле `omp/report.md`).

`FillColumn()` — заполняет результирующие массивы значениями и индексами строк для одного столбца
(листинг представлен в файле `omp/report.md`).

`MultiplyMatrices()` — основной алгоритм с TBB распараллеливанием:

- Реализует двухпроходную схему (описанную в разделе 4);
- Использует статическое разбиение столбцов на блоки;
- В каждом проходе создаются потоки, которые независимо обрабатывают свои диапазоны столбцов;
- Первый проход: параллельное вычисление количества ненулевых элементов для каждого столбца;
- Второй проход: параллельное заполнение результирующих массивов.

```cpp
// File: kotelnikova_a_double_matr_mult/stl/src/ops_stl.cpp
SparseMatrixCCS KotelnikovaATaskSTL::MultiplyMatrices(const SparseMatrixCCS &a, const SparseMatrixCCS &b) {
  SparseMatrixCCS result(a.rows, b.cols);

  const double epsilon = 1e-10;
  const unsigned int hardware_threads = std::thread::hardware_concurrency();
  const int num_threads = (hardware_threads > 0) ? static_cast<int>(hardware_threads) : 1;
  const int cols_per_thread = (b.cols + num_threads - 1) / num_threads;

  std::vector<int> col_start(b.cols, 0);
  std::vector<std::thread> threads;

  // Первый проход: подсчет ненулевых элементов
  for (int thread_idx = 0; thread_idx < num_threads; ++thread_idx) {
    int start_col = thread_idx * cols_per_thread;
    int end_col = std::min(start_col + cols_per_thread, b.cols);

    threads.emplace_back([&, start_col, end_col]() {
      for (int j = start_col; j < end_col; ++j) {
        std::vector<double> column = ComputeColumn(a, b, j);
        col_start[j] = CountNonZero(column, epsilon);
      }
    });
  }

  for (auto &thread : threads) {
    thread.join();
  }

  // Префиксная сумма для построения col_ptrs
  std::vector<int> col_ptr(b.cols + 1, 0);
  for (int j = 0; j < b.cols; ++j) {
    col_ptr[j + 1] = col_ptr[j] + col_start[j];
  }

  const int total_nnz = col_ptr[b.cols];
  result.values.resize(static_cast<size_t>(total_nnz));
  result.row_indices.resize(static_cast<size_t>(total_nnz));
  result.col_ptrs = col_ptr;

  // Второй проход: заполнение данных
  threads.clear();
  for (int thread_idx = 0; thread_idx < num_threads; ++thread_idx) {
    int start_col = thread_idx * cols_per_thread;
    int end_col = std::min(start_col + cols_per_thread, b.cols);

    threads.emplace_back([&, start_col, end_col]() {
      for (int j = start_col; j < end_col; ++j) {
        std::vector<double> column = ComputeColumn(a, b, j);
        FillColumn(column, epsilon, result.row_indices, result.values, col_ptr[j]);
      }
    });
  }

  for (auto &thread : threads) {
    thread.join();
  }

  return result;
}
```
