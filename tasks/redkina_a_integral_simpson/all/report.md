## 1. Контекст

Гибрид нужен, когда данные крупные и доступна **нескольким MPI-процессам**: сначала делится **глобальный** диапазон узлов
сетки, затем внутри процесса локальный кусок может обрабатываться несколькими асинхронными задачами (как в **STL**).
В **данной реализации** нет OpenMP/TBB внутри **`all`** — только **MPI** и **`std::async`** (см. `ops_all.cpp`).

## 2. Постановка задачи

См. **`seq/report.md`** и корневой **`report.md`**. Эталон — **SEQ**; согласованность между рангами обеспечивает
**`MPI_Allreduce`**.

## 3. Базовый алгоритм

Та же дискретизация Симпсона и линейная нумерация узлов **0 … total_points−1**.

## 4. Межпроцессная схема

- **`MPI_Comm_rank`**, **`MPI_Comm_size`** — определение **rank** и числа процессов **P**.
- Узлы делятся почти поровну: первые **`total_points % P`** рангов получают на один узел больше (см. формулы
  **`local_start`**, **`local_end`** в **`RunImpl`**).
- Глобальная сумма взвешенных значений собирается **`MPI_Allreduce(..., MPI_SUM, MPI_COMM_WORLD)`** — все ранги получают
  одинаковый **global_sum**; финальный множитель **(h_prod/3^d)** применяется после редукции.

## 5. Внутрипроцессная схема

**`ComputeLocalSumMPI(local_start, local_end, …)`:**

- **`num_threads = min(hardware_concurrency(), local_size)`**; при **0** из **`hardware_concurrency()`** подставляется **2**,
  но не больше длины локального диапазона.
- Если **`num_threads == 1`**, вызывается один **`ComputeRangeSum`** без **`std::async`**.
- Иначе локальный интервал режется, для каждого куска — **`std::async`**, сбор через **`future::get()`**.

**Итоговая конфигурация «работников»:** **`total_workers ≈ P × (число async на процесс)`**, но число **async** не фиксировано
явно — зависит от **local_size** и **`hardware_concurrency()`**.

## 6. Детали реализации

**Файлы:** `all/include/ops_all.hpp`, `all/src/ops_all.cpp`.

```cpp
// File: all/src/ops_all.cpp (фрагмент)
MPI_Comm_rank(MPI_COMM_WORLD, &rank);
MPI_Comm_size(MPI_COMM_WORLD, &world_size);
const double local_sum = ComputeLocalSumMPI(local_start, local_end, a_, h, weights, strides, func_, dim);
MPI_Allreduce(&local_sum, &global_sum, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);
```

**`MPI_Barrier`** в этой версии **не используется**; синхронизация результата — через **`MPI_Allreduce`**.
