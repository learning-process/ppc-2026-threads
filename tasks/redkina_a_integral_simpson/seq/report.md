# Последовательная версия (SEQ)

Реализация: `seq/src/ops_seq.cpp`.

## Идея

Сумма по составной формуле Симпсона записывается как взвешенная сумма значений функции во всех узлах **d**-мерной равномерной сетки: в каждом измерении **n_i** отрезков соответственно **n_i + 1** узлов.

## Обход сетки

Узлы перебираются в **лексикографическом порядке**: функция `AdvanceIndices` увеличивает индекс сначала по последнему измерению (оно меняется чаще всего), затем переносит переполнение к более старшим измерениям — как вложенные циклы с самым «быстрым» внутренним циклом по последней оси.

## Веса и результат

Для узла с многомерным индексом **(i_1, …, i_d)** одномерный вес по оси **k** задаётся правилами Симпсона:

| Условие | Вес |
|--------|-----|
| **i_k = 0** или **i_k = n_k** | 1 |
| внутренний узел, **i_k** нечётный | 4 |
| внутренний узел, **i_k** чётный | 2 |

Полный вес узла — произведение одномерных весов; к сумме добавляется **w_prod · f(x)**. Координата в узле: **x_k = a_k + i_k · h_k**, **h_k = (b_k − a_k) / n_k**.

После обхода всех узлов интеграл получается умножением суммы на множитель **(h₁·h₂·…·h_d) / 3^d**.

В коде произведение шагов накапливается в `h_prod`, знаменатель **3^d** — в `denominator`, итог: `result_ = (h_prod / denominator) * sum`.
