# TBB-версия

Реализация: `tbb/src/ops_tbb.cpp` (Intel oneTBB: `#include <oneapi/tbb.h>`).

## Идея

Используется **`tbb::parallel_reduce`** над диапазоном линейных индексов **`[0, total_points)`**, где `total_points` совпадает с числом узлов сетки (как в OMP: `strides[0] · (n[0] + 1)`). Библиотека сама дробит диапазон на поддиапазоны (`tbb::blocked_range<size_t>`), запускает работу на потоках и комбинирует частичные суммы бинарной операцией сложения.

## Предвычисления

Как в OMP/STL: таблицы весов Симпсона (`PrecomputeWeights`), шаги `h`, произведение `h_prod`, массив **strides** для перевода линейного индекса в многомерные координаты узла.

## Тело редукции

В лямбде для каждого поддиапазона заводятся буферы `indices` и `point`. Для каждого `linear_idx` из поддиапазона координаты восстанавливаются через `strides`, затем считается произведение весов по осям и добавляется **`w_prod * func_(point)`** к локальной сумме. Отдельной функции вида `ComputeNodeContribution` в коде нет — логика **встроена в лямбду**.

Функция объединения частичных сумм — стандартное сложение двух `double`. После редукции результат умножается на **(h_prod / 3^d)**.
