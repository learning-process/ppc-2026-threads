## Постановка задачи

Разработать программу, выполняющую свёртку целочисленного полутонового изображения с ядром Гаусса 3×3:

```cpp
static constexpr std::array<std::array<int, 3>, 3> kKernel = {{
    {1, 2, 1},
    {2, 4, 2},
    {1, 2, 1}
}};
static constexpr int kShift = 4;   // деление на 16
```

**Входные данные (`InType`):**

- ширина `width > 0`,
- высота `height > 0`,
- количество вертикальных полос `parts` (`1 ≤ parts ≤ width`),
- матрица пикселей `image[height][width]`.

**Выходные данные (`OutType`):**

- матрица того же размера, содержащая результат фильтрации.

**Ограничения:**

- границы изображения обрабатываются нулевым дополнением;
- выполняется проверка корректности размеров изображения;
- требуется инициализированная среда MPI.

## Описание алгоритма

Алгоритм использует двухуровневый параллелизм:

- MPI — распределение строк изображения между процессами;
- OpenMP — параллельная обработка строк внутри процесса.

Гибридная реализация сочетает межпроцессный параллелизм MPI
и внутрипроцессный параллелизм OpenMP.

MPI используется для распределения строк изображения между процессами,
а OpenMP — для параллельной обработки данных внутри каждого процесса.

### Основные этапы алгоритма

1. Исходное изображение распределяется между MPI-процессами по строкам.
2. Для каждого процесса вычисляются границы локального блока:

```cpp
int base = height / size;
int rem = height % size;
int start = rank * base + std::min(rank, rem);
int end = start + base + (rank < rem ? 1 : 0);
```

3. Каждый MPI-процесс независимо обрабатывает свой набор строк.
4. Внутри процесса цикл по строкам распараллеливается с помощью OpenMP.
5. Для обработки границ используется функция:

```cpp
GetPixelZero(...)
```

которая возвращает `0` при выходе за границы изображения.

6. Для каждого пикселя вычисляется свёртка с ядром Гаусса.
7. Результат нормализуется побитовым сдвигом:

```cpp
result = sum >> kShift;
```

8. Частичные результаты объединяются с помощью:

```cpp
MPI_Allgatherv(...)
```

### Ключевые элементы реализации

#### Обработка границ изображения

```cpp
int GetPixelZero(const std::vector<std::vector<int>> &img,
                 int row, int col,
                 int width, int height)
```

Функция возвращает `0`, если координаты выходят за пределы изображения.

---

### Распределение строк между MPI-процессами

```cpp
const int base = h / size;
const int rem = h % size;

const int start = (rank * base) + std::min(rank, rem);
const int end = start + base + ((rank < rem) ? 1 : 0);
```

---

### OpenMP-параллелизм

```cpp
#pragma omp parallel for default(none) \
    shared(local_flat, w, start, end, img, h)
```

---

### Сбор результатов

```cpp
MPI_Allgatherv(local_flat.data(),
               local_rows * w,
               MPI_INT,
               flat_result.data(),
               recv_counts.data(),
               displs.data(),
               MPI_INT,
               MPI_COMM_WORLD);
```

---

### Особенности реализации

- используется гибридная схема MPI + OpenMP;
- распределение выполняется по строкам изображения;
- применяется нулевой padding;
- обмен halo-областями не требуется;
- свёртка развёрнута вручную для повышения производительности;
- результаты собираются через `MPI_Allgatherv`.
