## 1. Контекст

Intel oneAPI Threading Building Blocks — это библиотека для создания параллельных программ на C++.
TBB ориентирована на использование задач, а не потоков напрямую, что позволяет достичь более
гибкого и динамического распределения нагрузки на многоядерных системах. В данной работе TBB
применяется для оптимизации блочной фильтрации изображений.

## 2. Постановка задачи

Задача состоит в реализации параллельной обработки изображения фильтром Гаусса с использованием
высокоуровневых абстракций TBB. Основное внимание уделяется эффективности работы с двумерными
диапазонами данных и автоматическому балансировщику нагрузки TBB.

## 3. Базовый алгоритм

Алгоритм реализует свертку 3x3. Последовательная логика подробно описана в `seq/report.md`.
В версии TBB сохраняется принцип блочности, но управление выполнением передается планировщику
библиотеки.

## 4. Схема распараллеливания

Для реализации параллелизма используется алгоритм `tbb::parallel_for` в сочетании с двумерным
диапазоном `tbb::blocked_range2d`:

1. Итерационное пространство определяется как прямоугольная область пикселей `[0, height) x [0, width)`.
2. Диапазон разбивается на блоки размером `block_height x block_width`.
3. TBB рекурсивно делит этот диапазон до тех пор, пока блоки не станут меньше или равны заданному размеру зерна.
4. Каждая задача обрабатывает свой подотрезок строк и столбцов.
5. Использование `blocked_range2d` позволяет TBB эффективно использовать кэш-память, обеспечивая локальность
   данных как по строкам, так и по столбцам.
6. Планировщик TBB использует стратегию "work-stealing", что гарантирует высокую загрузку всех ядер
   даже при неоднородной нагрузке.

## 5. Детали реализации

Основной фрагмент кода `RunImpl`:

```cpp
bool VdovinAGaussBlockTBB::RunImpl() {
  if (input_image_.empty() || output_image_.empty()) {
    return false;
  }

  int block_height = std::max(1, height_ / 4);
  int block_width = std::max(1, width_ / 4);

  tbb::parallel_for(tbb::blocked_range2d<int>(0, height_, block_height, 0, width_, block_width),
                    [this](const tbb::blocked_range2d<int> &range) {
    for (int py = range.rows().begin(); py < range.rows().end(); py++) {
      for (int px = range.cols().begin(); px < range.cols().end(); px++) {
        ApplyGaussianToPixel(py, px);
      }
    }
  });

  return true;
}
```

Здесь `ApplyGaussianToPixel` — потокобезопасная функция, так как каждый поток пишет в уникальные ячейки
памяти выходного вектора.
