## 1. Контекст

Решается задача численного вычисления трёхмерного интеграла методом прямоугольников. Последовательная версия служит базовым эталоном для оценки ускорения при использовании параллельных технологий (OpenMP, STL, TBB, All).

## 2. Постановка задачи

Входные данные - n — целое положительное число, количество разбиений по каждому измерению
Выходные данные - приближённое значение интеграла
Ограничения - n > 0 (проверяется в ValidationImpl), Размерность фиксирована - k_dim = 3

## 3. Базовый алгоритм

1 - Инициализация
   1.1 Получаем n из входных данных
   1.2 Вычисляем шаг сетки h = 1.0 / n
   1.3 Создаем массив индексов idx[k_dim] = {0, 0, 0}
   1.4 Создаем массив координат x[k_dim] = {0, 0, 0}
   1.5 Обнуляем сумму sum = 0.0
2 - Обход всех узлов сетки
   2.1 Для текущего набора индексов вычисляем координаты центров ячеек x[dim] = (idx[dim] + 0.5) * h
   2.2 Вычисляем значение подынтегральной функции exp(-(x^2 + y^2 + z^2))
   2.3 Значение добавляем к сумме
   2.4 Инкрементируем индексы с переносом разряда
     2.4.1 Увеличивается idx[0]
     2.4.2 Если idx[0] == n, сброс в 0 и увеличение idx[1]
     2.4.3 Если idx[1] == n, сброс в 0 и увеличение idx[2]
     2.4.4 Если idx[2] == n, цикл завершается
3 - Получение финального результата
   3.1 Объём ячейки volume = h^3
   3.2 Результат result = sum * volume

Сложност по времени - O(n^3) — прямо пропорционально количеству ячеек сетки
Сложность по памяти - O(1) - дополнительной памяти

## 4. Детали реализации

seq/include/ops_seq.hpp — объявление класса AfanasyevAIntegRectMethodSEQ
seq/src/ops_seq.cpp — реализация

ValidationImpl() - Проверяет n > 0. Возвращает false для недопустимых входных данных
PreProcessingImpl() - Не требует дополнительной подготовки, всегда true
RunImpl() - Выполняет основной алгоритм: обход сетки, суммирование, умножение на объём
PostProcessingImpl() - Не требует постобработки, всегда true

Граничные случаи: n <= 0 - валидация возвращает false, задача не выполняетс, n = 1 — корректно обрабатывается (одна итерация цикла)

```cpp
std::vector<int> idx(k_dim, 0);  // массив индексов (i,j,k)
std::vector<double> x(k_dim, 0.0);
double sum = 0.0;
bool done = false;

while (!done) {
  // вычисление координат центра ячейки
  for (int dim = 0; dim < k_dim; ++dim) {
    x[dim] = (static_cast<double>(idx[dim]) + 0.5) * h;
  }
  
  sum += ExampleIntegrand(x);  // накопление суммы
  
  // ручной инкремент индексов с переносом разряда
  for (int dim = 0; dim < k_dim; ++dim) {
    idx[dim]++;
    if (idx[dim] < n) break;
    idx[dim] = 0;
    if (dim == k_dim - 1) done = true;
  }
}
```
