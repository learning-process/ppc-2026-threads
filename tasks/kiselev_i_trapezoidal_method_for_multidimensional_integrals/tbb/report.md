# Метод трапеций для многомерных интегралов — TBB

## 1. Контекст

oneTBB-реализация использует task-based модель параллелизма и runtime-managed распределение задач.

---

## 2. Схема распараллеливания

Реализация использует:

```cpp
tbb::parallel_reduce
```

совместно с двумерным диапазоном:

```cpp
blocked_range2d<int, int>
```

Размер grain-блока:

```cpp
32 × 32
```

Каждый worker вычисляет локальную сумму, после чего выполняется reduction.

---

## 3. Детали реализации

Файлы:

- `tbb/include/ops_tbb.hpp`
- `tbb/src/ops_tbb.cpp`

Распределение нагрузки выполняется runtime scheduler-ом oneTBB.

Операция объединения результатов реализуется через:

```cpp
std::plus<>()
```
