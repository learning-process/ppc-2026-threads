# Метод трапеций для многомерных интегралов — OMP

## 1. Контекст

OpenMP-реализация использует loop-level распараллеливание вычислений в shared-memory среде.

---

## 2. Схема распараллеливания

Внешний цикл по координате `x` распараллеливается директивой:

```cpp
#pragma omp parallel for reduction(+ : result) default(none)
```

Shared-переменные:

- `nx`
- `ny`
- `hx`
- `hy`
- `x0`
- `y0`
- `func_type`

Использование `reduction` позволяет избежать гонок данных при накоплении суммы.

Используется статическое распределение итераций.

---

## 3. Детали реализации

Файлы:

- `omp/include/ops_omp.hpp`
- `omp/src/ops_omp.cpp`

Реализация сохраняет вычислительную схему последовательной версии, распараллеливая обработку строк сетки.

Дополнительная синхронизация через mutex или critical section не требуется.
