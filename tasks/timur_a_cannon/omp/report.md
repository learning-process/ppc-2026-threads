## 1. Контекст

Та же задача, что в SEQ, но циклы с большим объёмом работы выполняются **параллельно** с помощью OpenMP. Число потоков задаётся переменной `PPC_NUM_THREADS` (в системе также выставляется `OMP_NUM_THREADS`).

## 2. Постановка задачи

- **Вход:** кортеж `(b_size, A, B)` — размер блока `b_size` и две квадратные матрицы `A`, `B` размера `n×n` (`common/include/common.hpp`).
- **Выход:** матрица `C = A·B` того же размера `n×n`.
- **Ограничения:** `b_size > 0`; `n` кратно `b_size`; матрицы квадратные (проверяется в `ValidationImpl`).
- **Эталон для сравнения:** последовательная версия SEQ, режим `task_run`, один поток: **\(T_{seq} = 0.0499\) с** (матрицы `512×512`, блок `32×32`).

## 3. Базовый алгоритм

Используется **блочный алгоритм Кэннона** (та же математика, что в SEQ):

1. `grid_sz = n / b_size` — сетка блоков.
2. **Раскладка** `DistributeData`: для блока `(i,j)` фрагменты `A` и `B` берутся со сдвигом `(i+j) mod grid_sz`.
3. **Цикл шагов** `step = 0 … grid_sz-1`: для всех `(i,j)` — умножение блоков `BlockMultiplyAccumulate` с накоплением в `bl_c[i][j]`.
4. После шага (кроме последнего) — сдвиг блоков `RotateBlocksA` и `RotateBlocksB`.
5. **Сборка** `CollectResult` в полную матрицу `C`.

Сложность по времени: \(O(n^3)\). По памяти: \(O(n^2)\).

## 4. Схема распараллеливания

Параллелятся циклы по индексам сетки блоков `(i, j)` и отдельные циклы при сдвиге блоков:

- `DistributeData`, `CollectResult` — `#pragma omp parallel for collapse(2)` по `i` и `j`.
- Основной шаг Кэннона — такой же `parallel for collapse(2)` по всем блокам на шаге.
- `RotateBlocksA` / `RotateBlocksB` — `parallel for` по строке или столбцу сетки.

У каждого потока свой набор блоков для записи, поэтому **дополнительные mutex не нужны**. После параллельного цикла срабатывает **барьер** (конец области OpenMP), и только потом идёт сдвиг блоков.

Пример из кода:

```cpp
// File: omp/src/ops_omp.cpp
#pragma omp parallel for collapse(2) default(none) shared(bl_a, bl_b, bl_c, b_size, grid_sz)
for (int i = 0; i < grid_sz; ++i) {
  for (int j = 0; j < grid_sz; ++j) {
    BlockMultiplyAccumulate(bl_a[i][j], bl_b[i][j], bl_c[i][j], b_size);
  }
}
```
