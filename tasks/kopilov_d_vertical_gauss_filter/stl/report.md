## 1. Контекст

Данный отчет описывает реализацию параллельного фильтра Гаусса с использованием низкоуровневых примитивов
стандартной библиотеки C++ (`std::thread`). В отличие от OMP или TBB, здесь распределение нагрузки
и управление жизненным циклом потоков осуществляется вручную.

## 2. Постановка задачи

Необходимо разделить изображение на вертикальные полосы, назначить каждую полосу отдельному потоку
и собрать результат. Задача требует точного расчета границ данных, чтобы избежать наложений или пропусков.
Эталон корректности — версия SEQ.

## 3. Базовый алгоритм

Алгоритм применяет свертку 3x3. Вычисление нового значения пикселя требует доступа к восьми его соседям.
Поскольку мы только читаем исходные данные и записываем результат в новый массив, задача обладает
идеальным параллелизмом по данным.

## 4. Схема распараллеливания

В этой версии реализована ручная декомпозиция по ширине изображения:

1. **Разбиение**: Ширина `width` делится на количество потоков `num_threads`.
2. **Балансировка**: Чтобы обработать остаток (если ширина не делится нацело), первым потокам
   назначается на один столбец больше.
3. **Потокобезопасность**: Синхронизация через `std::mutex` не используется. Каждый поток пишет
   строго в свою область памяти (свой набор столбцов), что исключает состояние гонки.
4. **Управление**: Потоки запускаются в цикле, сохраняются в вектор, а затем ожидаются основным потоком.

## 5. Детали реализации

Файлы: `stl/include/ops_stl.hpp`, `stl/src/ops_stl.cpp`.

Критически важно, чтобы вызов `.join()` происходил в отдельном цикле. Если вызывать его сразу после
создания потока, программа станет последовательной.

```cpp
// Фрагмент из stl/src/ops_stl.cpp
for (int thread_idx = 0; thread_idx < actual_threads_count; ++thread_idx) {
  int end_column = start_column + cols_per_thread + (thread_idx < remainder_cols ? 1 : 0);
  threads.emplace_back([&, start_column, end_column]() {
    for (int col_idx = start_column; col_idx < end_column; ++col_idx) {
      for (int row_idx = 0; row_idx < height; ++row_idx) {
        result[(row_idx * width) + col_idx] = ApplyGaussKernel(matrix, col_idx, row_idx, width, height);
      }
    }
  });
  start_column = end_column;
}

for (auto &thread_obj : threads) {
  thread_obj.join();
}
```

## 6. Проверка корректности

Результаты STL-версии проверялись через `tests/functional/main.cpp`. Программа успешно проходит
тесты на матрицах 1x1, 2x2 и 4x4. Попиксельное сравнение с SEQ подтвердило отсутствие ошибок
в расчете границ полос.
