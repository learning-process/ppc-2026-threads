## 1. Контекст

Последовательная эталонная реализация умножения квадратных плотных матриц
с использованием оптимизированного алгоритма Штрассена. Элементы матриц - типа double.
Алгоритм Штрассена имеет сложность O(n^2.807), что быстрее стандартного O(n^3).
Данная версия служит baseline для оценки корректности и измерения ускорения
параллельных реализаций (OMP, TBB, STL, ALL).

## 2. Постановка задачи

### 2.1. Входные данные

Входные данные - вектор `std::vector<double>`:

- [0] - размер матрицы n (n > 0)
- [1 ... n^2] - матрица A (построчно)
- [n^2+1 ... 2n^2] - матрица B (построчно)

### 2.2. Выходные данные

Выходные данные - вектор `std::vector<double>`:

- [0] - размер n
- [1 ... n^2] - матрица C = A x B

### 2.3. Ограничения

- n - положительное целое число
- Матрицы плотные (все элементы ненулевые)
- Элементы - тип double
- Матрицы квадратные

### 2.4. Крайние случаи

- Пустые входные данные -> valid_data = false, возврат пустого результата
- n <= 0 -> valid_data = false, возврат пустого результата
- Неверный размер вектора -> valid_data = false, возврат пустого результата
- n = 1 -> стандартное умножение
- n нечётное -> переключение на стандартное умножение
- n <= 64 -> стандартное умножение (без рекурсии)

## 3. Базовый алгоритм

### 3.1. Алгоритм Штрассена (рекурсивный)

Шаг 1: Базовый случай - если n <= 64 или n нечётное, использовать стандартное умножение.

Шаг 2: Разбить матрицы A и B на 4 подматрицы размером n/2 x n/2.

Шаг 3: Вычислить 7 промежуточных матриц P1...P7.

Шаг 4: Вычислить подматрицы результата C11, C12, C21, C22.

Шаг 5: Объединить подматрицы в итоговую матрицу C.

### 3.2. Стандартное умножение (базовый случай)

```cpp
for (int i = 0; i < n; ++i)
    for (int j = 0; j < n; ++j)
        for (int k = 0; k < n; ++k)
            C[i][j] += A[i][k] * B[k][j];
```

### 3.3. Асимптотика

- Время (Штрассен): O(n^2.807)
- Время (стандартное): O(n^3)
- Память: O(n^2)

### 3.4. Критерий корректности

`|C_alg[i][j] - C_standard[i][j]|` < 1e-6 для всех i, j

### 4.2. Pipeline обработки

- `ValidationImpl()`: всегда возвращает true
- `PreProcessingImpl()`: проверка данных, извлечение матриц A и B
- `RunImpl()`: выбор стратегии умножения, вычисление
- `PostProcessingImpl()`: формирование выходного вектора

### 4.3. PreProcessingImpl (ключевой код)

```cpp
bool MorozovaSStrassenMultiplicationSEQ::PreProcessingImpl() {
    if (GetInput().empty()) {
        valid_data_ = false;
        return true;
    }
    double size_val = GetInput()[0];
    if (size_val <= 0.0) {
        valid_data_ = false;
        return true;
    }
    int n = static_cast<int>(size_val);
    if (GetInput().size() != 1 + (2 * (size_t)n * (size_t)n)) {
        valid_data_ = false;
        return true;
    }
    valid_data_ = true;
    n_ = n;
    a_ = Matrix(n_);
    b_ = Matrix(n_);
    int idx = 1;
    for (int i = 0; i < n_; ++i)
        for (int j = 0; j < n_; ++j)
            a_(i, j) = GetInput()[idx++];
    for (int i = 0; i < n_; ++i)
        for (int j = 0; j < n_; ++j)
            b_(i, j) = GetInput()[idx++];
    return true;
}
```

### 4.4. RunImpl (ключевой код)

```cpp
bool MorozovaSStrassenMultiplicationSEQ::RunImpl() {
    if (!valid_data_) return true;
    const int leaf_size = 64;
    if (n_ <= leaf_size) {
        c_ = MultiplyStandard(a_, b_);
    } else {
        c_ = MultiplyStrassen(a_, b_, leaf_size);
    }
    return true;
}
```
