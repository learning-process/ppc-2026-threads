## Постановка задачи

Вычислить среднее арифметическое элементов вектора целых чисел.

## Описание алгоритма

### SEQ

Сумма элементов / количество элементов.

### MPI

1. Разделение вектора между процессами
2. Локальная сумма
3. MPI_Reduce
4. Вычисление среднего на процессе 0

## Результаты

Функциональные тесты пройдены для размеров: 10, 100, 1000 элементов.

Тесты производительности (1,000,000 элементов):

- SEQ: ~1.95 мс
- MPI (4 процесса): ~2.10 мс
