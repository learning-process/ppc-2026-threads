## 1. Контекст

Версия на базе Intel oneTBB реализует ту же схему разделения изображения на полосы, что и OMP, но использует механизмы
планирования задач TBB. Это позволяет интегрировать решение в экосистему TBB 
и использовать `task_arena` для изоляции вычислительной нагрузки.

## 2. Постановка задачи

Краткая постановка: маркировка связных областей. SEQ-версия ([seq/report.md](../seq/report.md)) — baseline.

## 3. Базовый алгоритм

Последовательная версия использует BFS. В TBB-версии изображение делится на $N$ горизонтальных полос (по числу потоков),
которые обрабатываются параллельно, после чего выполняется слияние через DSU.

## 4. Схема распараллеливания

- **Примитив:** Использован `tbb::parallel_for` в сочетании с `tbb::task_arena`.
- **Диапазон:** Диапазон работы `parallel_for` установлен от `0` до `num_threads`. Таким образом, каждая итерация
  соответствует одному логическому потоку (тикету).
- **Grainsize:** Не задается явно, так как диапазон ограничен числом потоков.
- **Partitioner:** Использован `tbb::simple_partitioner()`. Это гарантирует, что диапазон не будет дробиться дальше и
  каждый поток получит ровно одну порцию работы (одну или несколько полос в зависимости от разбиения).
- **Контроль конкуренции:** Число потоков ограничивается явно через создание `tbb::task_arena arena(num_threads)` и
  выполнение кода внутри `arena.execute`. Это более жесткий и локальный способ контроля по сравнению с глобальными
  настройками.

## 5. Детали реализации

- **Файлы:** `tbb/include/ops_tbb.hpp`, `tbb/src/ops_tbb.cpp`
- **Распределение нагрузки:** Внутри `parallel_for` каждый поток (индекс `tid`) самостоятельно вычисляет свои границы
  строк `row_start` и `row_end` на основе общего количества потоков и высоты изображения.
- **Слияние:** После выхода из `arena.execute` выполняется последовательное слияние строк (`MergeHorizontal`) и границ
  между полосами (`MergeBoundaries`) через DSU.

```cpp
tbb::task_arena arena(num_threads);
arena.execute([&] {
  tbb::parallel_for(tbb::blocked_range<int>(0, num_threads), [&](const tbb::blocked_range<int> &range) {
    for (int tid = range.begin(); tid < range.end(); ++tid) {
      const int row_start = (tid * height) / num_threads;
      const int row_end = ((tid + 1) * height) / num_threads;
      const int label_offset = tid * max_labels_per_thread;
      ScanStripe(row_start, row_end, width, label_offset, image, local_labels);
    }
  }, tbb::simple_partitioner());
});
```
