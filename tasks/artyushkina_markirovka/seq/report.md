## 1. Контекст

Исходная задача — маркировка связных компонент на бинарном изображении, где чёрные пиксели (значение 0) соответствуют объектам, а белые (значение 255) — фону.

Последовательная версия служит эталоном корректности и базой для измерения ускорения параллельных реализаций. Алгоритм использует 8-связность: пиксели считаются связанными, если они соприкасаются по стороне или углу. Это позволяет корректно обрабатывать диагональные соединения.

SEQ-реализация использует обход в ширину (BFS) для каждой компоненты, что гарантирует правильную маркировку всех связных областей.

## 2. Постановка задачи

**Входные данные**: вектор `std::vector<uint8_t>`, где:
- `[0]` — количество строк `rows`
- `[1]` — количество столбцов `cols`
- `[2..]` — значения пикселей: `0` (объект) или `255` (фон)

**Выходные данные**: вектор `std::vector<uint8_t>`, где:
- `[0]` — `rows`
- `[1]` — `cols`
- `[2..]` — метки компонент (`0` — фон, `1..N` — уникальные метки объектов)

## 3. Базовый алгоритм

SEQ реализация использует обход в ширину (BFS). Для каждого непомеченного пикселя-объекта запускается BFS, который рекурсивно помечает всю связную компоненту уникальным номером.

```
для каждого пикселя (i, j):
    если пиксель является объектом и не помечен:
        current_label++
        очередь.push((i, j))
        метки[i][j] = current_label
        
        пока очередь не пуста:
            (x, y) = очередь.pop()
            для каждого из 8 соседей (dx, dy):
                если сосед является объектом и не помечен:
                    метки[x+dx][y+dy] = current_label
                    очередь.push((x+dx, y+dy))
```

Временная сложность: O(rows * cols). Затраты памяти: O(rows * cols) для хранения меток и очереди BFS.

## 4. Детали реализации

**Файлы реализации:**
- `seq/include/ops_seq.hpp` — объявление класса `MarkingComponentsSEQ`
- `seq/src/ops_seq.cpp` — реализация BFS-алгоритма

**Ключевые методы:**
- `BFS(int start_i, int start_j, int label)` — обход компоненты в ширину
- `IsValidNeighbor(int i, int j, const NeighborOffset &offset)` — проверка выхода за границы
- `ProcessNeighbor(...)` — обработка соседнего пикселя

**Массив направлений для 8 соседей:**
```cpp
const std::array<NeighborOffset, 8> kNeighbors = {{
    {-1, -1, true, false, true, false},  // верхний-левый
    {-1, 0, true, false, false, false},  // верхний
    {-1, 1, true, false, false, true},   // верхний-правый
    {0, -1, false, false, true, false},  // левый
    {0, 1, false, false, false, true},   // правый
    {1, -1, false, true, true, false},   // нижний-левый
    {1, 0, false, true, false, false},   // нижний
    {1, 1, false, true, false, true}     // нижний-правый
}};
```

**Реализация BFS:**
```cpp
void MarkingComponentsSEQ::BFS(int start_i, int start_j, int label) {
  std::queue<std::pair<int, int>> q;
  q.emplace(start_i, start_j);
  labels_[start_i][start_j] = label;

  while (!q.empty()) {
    auto [i, j] = q.front();
    q.pop();

    for (const auto &offset : kNeighbors) {
      ProcessNeighbor(i, j, offset, label, q);
    }
  }
}
```

**Проверка соседа:**
```cpp
bool MarkingComponentsSEQ::IsValidNeighbor(int i, int j, const NeighborOffset &offset) const {
  if (offset.check_i_min && i <= 0) return false;
  if (offset.check_i_max && i >= rows_ - 1) return false;
  if (offset.check_j_min && j <= 0) return false;
  if (offset.check_j_max && j >= cols_ - 1) return false;
  return true;
}

void MarkingComponentsSEQ::ProcessNeighbor(int i, int j, const NeighborOffset &offset,
                                           int label, std::queue<std::pair<int, int>> &q) {
  if (!IsValidNeighbor(i, j, offset)) return;

  int ni = i + offset.di;
  int nj = j + offset.dj;

  const auto &input = GetInput();
  size_t idx = (ni * cols_ + nj) + 2;

  if (input[idx] == 0 && labels_[ni][nj] == 0) {
    labels_[ni][nj] = label;
    q.emplace(ni, nj);
  }
}
```

**Основной цикл:**
```cpp
bool MarkingComponentsSEQ::RunImpl() {
  const auto &input = GetInput();
  if (input.size() < 2 || rows_ == 0 || cols_ == 0) return false;

  int current_label = 1;

  for (int i = 0; i < rows_; ++i) {
    for (int j = 0; j < cols_; ++j) {
      size_t idx = (i * cols_ + j) + 2;
      if (input[idx] == 0 && labels_[i][j] == 0) {
        BFS(i, j, current_label);
        ++current_label;
      }
    }
  }
  return true;
}
```
