### 1. Введение
**Мотивация.** Сортировка является базовой операцией для последующих алгоритмов поиска и обработки данных. Сортировка Шелла улучшает вставки за счет работы с разреженными подпоследовательностями, а простое слияние обеспечивает корректную сборку результата.  
**Проблематика.** Реализация должна быть корректной для отрицательных значений и дубликатов, а также не использовать параллельные подходы.  
**Ожидаемый результат.** Получить устойчиво корректную последовательную реализацию, проходящую функциональные и performance тесты.

### 2. Постановка задачи
**Задано:** вектор целых чисел.  
**Требуется:** отсортировать данные по возрастанию, применив сортировку Шелла для двух частей и выполнить простое слияние.  
**Входные данные:** `InType = std::vector<int>`.  
**Выходные данные:** `OutType = std::vector<int>` — отсортированный вектор.

### 3. Последовательный алгоритм
1. Входной массив копируется в выходной.  
2. Массив делится на левую и правую половины.  
3. Каждая половина сортируется сортировкой Шелла: последовательность шагов `gap = n/2, n/4, ..., 1`, внутри каждого шага выполняются вставки по подпоследовательностям.  
4. Две отсортированные части сливаются простым линейным слиянием в итоговый массив.  
5. Результат проверяется на упорядоченность.

### 4. Схема распараллеливания
Отсутствует. Реализация выполнена исключительно в последовательном режиме (seq).
