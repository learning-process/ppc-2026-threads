## 1. Введение

В данной работе рассматривается последовательная реализация алгоритма сортировки Шелла
с последующим слиянием двух отсортированных подмассивов. Эта реализация служит эталоном
для проверки корректности параллельных версий и базой для расчёта ускорения.

---

## 2. Постановка задачи

Требуется отсортировать вектор целых чисел произвольного размера по неубыванию.

### Условия корректности

- Результат является перестановкой исходного массива, упорядоченной по неубыванию.
- Алгоритм корректно работает на пустом массиве и массиве из одного элемента.

---

## 3. Описание алгоритма

Алгоритм разделяет массив на две половины, сортирует каждую из них с помощью сортировки
Шелла, затем сливает в итоговый массив через функцию `Merge`.

- **Фаза сортировки**: каждый подмассив сортируется `BaseShellSort` независимо.
- **Фаза слияния**: два отсортированных подмассива объединяются за один проход.

### Вычислительная сложность

- Сортировка Шелла: O(N log² N)
- Слияние: O(N)
- Дополнительная память: O(N) — нужны временные копии левой и правой частей.

---

## 4. Детали реализации

Файлы:

- `seq/include/ops_seq.hpp`
- `seq/src/ops_seq.cpp`

Пайплайн:

- **ValidationImpl** — проверяет, что входная структура не пуста.
- **PreProcessingImpl** — копирует входной вектор в выходной, сортировка выполняется in-place.
- **RunImpl** — запускает `ShellSortMerge`.
- **PostProcessingImpl** — не требует дополнительных действий.

Вектор делится на две части: левую `[0, mid)` и правую `[mid, n)`. Каждая сортируется
отдельно, после чего вызывается слияние. Использование вспомогательных векторов `left`
и `right` упрощает реализацию слияния.

## Приложение

```cpp
void LitvyakovDShellSortSEQ::ShellSortMerge(std::vector<int> &vec) {
  if (vec.size() <= 1) {
    return;
  }
  std::size_t mid = vec.size() / 2;
  std::vector<int> left(vec.begin(),
                        vec.begin() + static_cast<int64_t>(mid));
  std::vector<int> right(vec.begin() + static_cast<int64_t>(mid),
                         vec.end());
  BaseShellSort(left);
  BaseShellSort(right);
  Merge(left, right, vec);
}
```

Массив делится пополам, каждая половина сортируется независимо, затем происходит слияние.
