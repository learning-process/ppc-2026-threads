## Схема распараллеливания

OpenMP tasks применяются для рекурсивного распараллеливания quicksort-подзадач.
Для листовых диапазонов используется эффективная локальная сортировка.

## Реализация

- Файлы: `omp/include/ops_omp.hpp`, `omp/src/ops_omp.cpp`
- Ограничивается глубина распараллеливания, чтобы снизить overhead.
