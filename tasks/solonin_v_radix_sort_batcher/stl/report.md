# STL: Поразрядная сортировка для целых чисел с четно-нечетным слиянием Бэтчера

## Описание реализации

### Схема параллелизма

Используются `std::thread` для параллельной сортировки блоков. Каждый поток работает со своим независимым блоком данных.

```cpp
for (int i = 0; i < nthreads; ++i) {
  blocks.emplace_back(GetInput().begin() + lo, GetInput().begin() + hi);
  int block_idx = static_cast<int>(blocks.size()) - 1;
  workers.emplace_back([&blocks, block_idx]() {
    for (size_t pos_idx = 0; pos_idx < sizeof(int); ++pos_idx) {
      SortByDigit(blocks[block_idx], pos_idx);
    }
  });
}
for (auto &w : workers) w.join();
```

### Ручное разбиение

Диапазон `[0, n)` делится на `nthreads` равных частей. Каждый поток получает свой индекс блока по значению (захват `block_idx` по значению в лямбде).

### Синхронизация

- Нет общих данных между потоками на этапе сортировки
- `join()` вызывается для каждого потока после запуска всех
- Слияние Бэтчера выполняется последовательно в отдельных потоках

### Карта поток → данные

| Поток | Блок |
|---|---|
| 0 | `[0, chunk)` |
| 1 | `[chunk, 2*chunk)` |
| ... | ... |
| n-1 | `[(n-1)*chunk, n)` |

## Выводы

STL-версия использует ручное управление потоками. Накладные расходы на создание потоков компенсируются параллельной сортировкой на больших данных.
