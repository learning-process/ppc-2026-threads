# SEQ: Поразрядная сортировка для целых чисел с четно-нечетным слиянием Бэтчера

## Постановка задачи

Последовательная реализация поразрядной сортировки массива целых чисел. Служит эталонной базой для сравнения с параллельными реализациями.

## Описание реализации

### Алгоритм

Реализована LSD-поразрядная сортировка с 4 проходами по байтам 32-битного int. Каждый проход — устойчивая сортировка подсчётом по текущему байту (256 корзин).

```cpp
void RadixSortBatcherSEQ::SortByDigit(std::vector<int> &data, size_t pos) {
  const size_t k_base = 256;
  std::vector<int> freq(k_base, 0);
  bool last = (pos == sizeof(int) - 1ULL);
  for (int v : data) {
    int bv = (v >> (pos * 8ULL)) & 0xFF;
    if (last) bv ^= 0x80;  // обработка знакового бита
    freq[bv]++;
  }
  // префиксные суммы + перестановка
}
```

Ключевой момент: при обработке старшего байта применяется XOR с 0x80, что инвертирует знаковый бит и позволяет корректно отсортировать отрицательные числа без специальной обработки.

### ValidationImpl
Проверяет что входной массив непустой.

### PreProcessingImpl / PostProcessingImpl
Пустые — данные уже в `GetInput()` / `GetOutput()`.

### RunImpl
Для массива размером ≤ 1 возвращает копию входа. Иначе — 4 прохода `SortByDigit`.

## Корректность

Функциональные тесты покрывают:
- пустой массив (ValidationImpl → false)
- массив из 1 элемента
- уже отсортированный массив
- массив в обратном порядке
- массив с отрицательными числами
- массив с повторяющимися элементами

Эталон проверки: `std::sort`.

## Производительность

Сложность: O(n × k), где k = 4 (число байт в int). Линейная по размеру входа.

| Размер (N) | Время (мс) |
|---|---|
| 100 000 | ~15 |
| 1 000 000 | ~150 |
