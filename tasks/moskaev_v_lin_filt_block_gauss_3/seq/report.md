# Линейная фильтрация блоков – фильтр Гаусса 3×3 — SEQ

- Student: Москаев Владимир Александрович

## 1. Контекст

Решается задача блочной фильтрации изображения ядром Гаусса 3×3.
Фильтрация Гаусса применяется для размытия изображения, подавления шума,
уменьшения детализации. Блочная обработка позволяет обрабатывать большие
изображения, не загружая целиком в память, и естественно распараллеливается.

Последовательная версия служит эталоном корректности и производительности
для всех параллельных реализаций (OpenMP, TBB, STL, MPI+threads).

## 2. Постановка задачи

**Входные данные:**
- width (int) — ширина изображения в пикселях
- height (int) — высота изображения в пикселях
- channels (int) — число цветовых каналов (1 — серое, 3 — RGB)
- block_size (int) — размер блока (фиксирован 64)
- image_data (vector<uint8_t>) — пиксели в формате row-major, каналы interleaved

**Выходные данные:**
- vector<uint8_t> — отфильтрованное изображение той же размерности

**Ядро Гаусса 3×3 (нормированное):**

    1/16  2/16  1/16
    2/16  4/16  2/16
    1/16  2/16  1/16

В коде задано как константа kGaussianKernel из 9 элементов.

**Ограничения:**
- Вектор входных данных не должен быть пустым
- Размер изображения не менее 1×1
- Границы изображения обрабатываются симметричным отражением (clamp)

**Крайние случаи:**
- Изображение 1×1 — фильтр не применяется, выход равен входу
- Блоки у правой и нижней границы меньше 64×64 — обрабатываются корректно
  через std::min

## 3. Базовый алгоритм

**Пошаговое описание:**

1. **Разбиение на блоки:** изображение делится на блоки размером 64×64.
   Последние блоки по правому и нижнему краю могут быть меньше.

2. **Копирование блока с отступами (halo):** создаётся временный буфер
   с добавлением +1 пикселя со всех сторон. Пиксели за границей изображения
   заполняются ближайшими граничными через std::clamp.

   Фрагмент функции CopyBlockWithPadding:

```cpp
for (int row = -1; row <= current_block_height; ++row) {
    for (int col = -1; col <= current_block_width; ++col) {
        int src_y = std::clamp(block_y + row, 0, height - 1);
        int src_x = std::clamp(block_x + col, 0, width - 1);
        // копирование пикселя в буфер со сдвигом +1
    }
}
```

3. **Фильтрация блока:** для каждого внутреннего пикселя вычисляется свёртка с ядром Гаусса.

```cpp
   Фрагмент функции ApplyGaussianFilterToBlock:
   for (int row = 0; row < inner_height; ++row) {
       for (int col = 0; col < inner_width; ++col) {
           for (int channel = 0; channel < channels; ++channel) {
               float sum = 0.0f;
               for (int ky = -1; ky <= 1; ++ky)
                   for (int kx = -1; kx <= 1; ++kx)
                       sum += input[idx] * kGaussianKernel[ky*3 + kx];
               output[out_idx] = static_cast<uint8_t>(std::round(sum));
           }
       }
   }
```

4. **Запись результата:** обработанный блок копируется в выходное изображение.

**Асимптотика по времени:** O(W × H × 9) — 9 операций на каждый пиксель.

**Асимптотика по памяти:** O(W × H) для хранения входного и выходного изображений
плюс O(block_size²) для временных буферов.
